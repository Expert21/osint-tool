# OSINT Tool

A comprehensive, modular OSINT (Open Source Intelligence) CLI tool to search for targets (individuals or companies) across social media platforms and web search engines.

## Usage

```bash
python main.py --target "Target Name" --type [individual|company]
```

## Modules

- **Search Engines**: Google, Bing, DuckDuckGo
- **Social Media**: LinkedIn, Twitter, Instagram
